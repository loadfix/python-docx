Watermark
=========

.. currentmodule:: docx.watermark

The read side of text and image watermarks.
``Section.add_text_watermark()`` and ``Section.add_image_watermark()``
create watermarks; the :class:`.Watermark` object below exposes the
properties of an existing watermark.
