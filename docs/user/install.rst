.. _install:

Installing
==========

.. note:: python-docx versions 0.3.0 and later are not API-compatible with
   prior versions.

|docx| is hosted on PyPI, so installation with ``pip`` is straightforward::

    pip install python-docx

|docx| depends on the ``lxml`` package. ``pip`` will install it automatically
along with any other runtime dependencies.


Dependencies
------------

* Python 3.9+
* lxml >= 4.9.1
